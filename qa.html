Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHtml returns all texts with html tags that is inside of element

innerText gives back all texts that inside an element and its childs also

textContent has all text inside element and its childs, including hidden text, but it does not interpret HTML tags

Question-3: What is event delegation in the DOM?

Event delegation is a technique where an event listener is added to a parent element, allowing it to handle events for its child elements. This is useful for handling many items or dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling occurs when an event starts at the target element and then propagates upwards through its parent elements, reaching the top of the DOM hierarchy.
